# 前端接口消费文档

## 代理配置

Vite 开发服务器自动代理 `/api/*` → `http://127.0.0.1:8000`，直接写相对路径即可：

```javascript
fetch('/api/xxx')  // 自动转发到后端
```

## 一、对话与检索

### `POST /api/chat/stream` — SSE 流式对话

```javascript
const response = await fetch('/api/chat/stream', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: '如何更换滤网',
    history: [{ role: 'user', content: '...' }, { role: 'assistant', content: '...' }],
    session_id: 'sess_abc',
    user_context: { user_id: '1004', city: '北京', month: '2026-06' },
    mode: 'chat',   // chat | chat_only
  }),
})

// SSE 流式读取
const reader = response.body.getReader()
const decoder = new TextDecoder()
let buf = ''
while (true) {
  const { done, value } = await reader.read()
  buf += decoder.decode(value, { stream: true })
  const lines = buf.split('\n')
  buf = lines.pop() || ''
  for (const line of lines) {
    if (line.startsWith('data: ')) {
      const text = line.slice(6)
      if (text.startsWith('[ERROR]')) throw new Error(text.slice(8))
      if (text.trim()) content += text  // 逐块追加渲染
    }
  }
  if (done) break
}
```

### `POST /api/rag/search` — 知识库检索

```javascript
const resp = await fetch('/api/rag/search', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ query: 'WIFI设置' }),
})
const { data } = await resp.json()
// data.results: [{ content, score, source, category, metadata }]
```

## 二、管理端接口

> 所有管理端接口均需传入 `password` 字段，密码错误返回 403。

### `POST /api/admin/knowledge` — 获取知识库完整结构

```javascript
const resp = await fetch('/api/admin/knowledge', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ password: 'xxx' }),
})
const { data } = await resp.json()
// data: { chunk_count, category_count, tracked_file_count, collection_name, categories }
// categories: [{ name, file_count, total_chunks, files: [{ name, source, chunk_count, chunks }] }]
```

### `POST /api/admin/upload` — 上传文档

```javascript
const form = new FormData()
form.append('password', 'xxx')
form.append('category', '产品FAQ')
form.append('file', fileInput.files[0])  // PDF 或 TXT

const resp = await fetch('/api/admin/upload', { method: 'POST', body: form })
const data = await resp.json()
```

### `POST /api/admin/delete` — 删除文档

```javascript
const resp = await fetch('/api/admin/delete', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    password: 'xxx',
    source: 'C:\\path\\to\\file.pdf',  // 完整路径，从 knowledge 接口获取
  }),
})
```

### `POST /api/admin/category` — 更改文档分类

```javascript
const resp = await fetch('/api/admin/category', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    password: 'xxx',
    source: 'C:\\path\\to\\file.pdf',
    category: '新分类名',
  }),
})
```

### `POST /api/admin/chunk/update` — 编辑单个 Chunk

```javascript
const resp = await fetch('/api/admin/chunk/update', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    password: 'xxx',
    chunk_id: 'chunk-uuid-from-knowledge-api',
    content: '新的文本内容...',
  }),
})
```

## 三、会话与画像

| 操作 | 方法 | 路径 |
|---|---|---|
| 列出所有会话 | GET | `/api/sessions` |
| 获取会话历史 | GET | `/api/session/{session_id}` |
| 删除会话 | DELETE | `/api/session/{session_id}` |
| 保存画像 | POST | `/api/profile` → `{ user_id, city, gender, age, preferences }` |
| 获取画像 | GET | `/api/profile/{user_id}` |

## 通用约定

**响应格式：**
```json
{ "success": true | false, "message": "说明", "data": { ... } | null }
```

**错误处理：**
```javascript
const data = await resp.json()
if (!resp.ok || !data.success) {
  throw new Error(data.message || '请求失败')
}
```
