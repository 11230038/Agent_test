# 前端接口消费文档

## 后端地址

开发环境通过 Vite 代理转发，无需额外配置：

```
/api/*  →  http://127.0.0.1:8000/api/*
```

## 1. 智能对话 / 仅聊天

### 请求

```javascript
const response = await fetch('/api/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: '用户输入的内容',
    history: [                         // 最近 15 条
      { role: 'user', content: '...' },
      { role: 'assistant', content: '...' },
    ],
    session_id: 'sess_abc123',
    user_context: {
      user_id: '1004',
      city: '北京',
      month: '2026-06',
    },
    mode: 'chat',                      // chat | chat_only | report
  }),
})
```

### 响应

```json
{
  "success": true,
  "message": "请求成功",
  "data": {
    "answer": "回答内容...",
    "trace_id": "a1b2c3d4e5f6",
    "sources": [
      { "source": "维护保养.txt", "category": "维护保养", "score": 0.714 }
    ]
  }
}
```

### 前端处理

```javascript
const data = await response.json()

if (!response.ok || data.success === false) {
  throw new Error(data.message)  // 展示通用错误文案
}

messages.value.push({
  role: 'assistant',
  content: data.data?.answer || '暂时没有获取到回答',
})

// data.data.sources  可展示参考来源
// data.data.trace_id 可用于问题追踪
```

### 三种模式

| mode | 说明 | 前端标题 |
|---|---|---|
| `chat` | 智能问答，可使用所有工具和知识库 | 扫地机器人智能客服 |
| `chat_only` | 纯聊天，无工具无知识库 | 小扫 · 聊聊天 |
| `report` | 报告生成模式 | 扫地机器人智能客服 |

## 2. 知识库检索

### 请求

```javascript
const response = await fetch('/api/rag/search', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ query: 'WIFI设置' }),
})
```

### 响应

```json
{
  "success": true,
  "data": {
    "query": "WIFI设置",
    "total": 2,
    "results": [
      {
        "content": "文档片段内容...",
        "score": 1.128,
        "source": "扫拖一体机器人100问.txt",
        "category": "扫拖一体",
        "metadata": { ... }
      }
    ]
  }
}
```

### 前端渲染

```javascript
const data = await response.json()
searchResults.value = data.data?.results || []
// 每条结果：
//   content  → 文本片段
//   score    → 相关性分值（相对值，越高越相关）
//   source   → 来源文件名
//   category → 文档分类标签
```

## 3. 知识库状态

```javascript
fetch('/api/rag/status').then(r => r.json())
```

返回 `chunk_count`、`categories`、`cache` 命中率、`embedding` 可用性。

## 4. 会话接口

### 获取历史会话列表

```javascript
fetch('/api/sessions').then(r => r.json())
// → { sessions: [{ session_id, msg_count, last_active }] }
```

### 获取指定会话历史

```javascript
fetch('/api/session/sess_abc').then(r => r.json())
// → { session_id, history: [{ role, content }] }
```

### 删除会话

```javascript
fetch('/api/session/sess_abc', { method: 'DELETE' })
```

## 5. 用户画像

### 获取画像

```javascript
fetch('/api/profile/1004').then(r => r.json())
// → { profile: { user_id, city, gender, age, preferences, updated_at } }
```

### 保存画像

```javascript
fetch('/api/profile', {
  method: 'POST',
  body: JSON.stringify({
    user_id: '1004',
    city: '上海',
    gender: '男',
    age: 35,
    preferences: { robot_type: '扫拖一体' },
  }),
})
```

## 6. 反馈

### 提交反馈

```javascript
fetch('/api/feedback', {
  method: 'POST',
  body: JSON.stringify({
    session_id: 'sess_abc',
    rating: 'like',        // like | dislike
    comment: '回答很准确',
  }),
})
```

### 查看统计

```javascript
fetch('/api/feedback/stats').then(r => r.json())
// → { total, likes, dislikes, recent }
```
