## 核心文件

### `vite.config.js` — 构建 + 代理

```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://127.0.0.1:8000',  // 开发时转发到后端
      changeOrigin: true,
    },
  },
}
```

### `src/main.js` — 入口

```javascript
import { createApp } from 'vue'
import App from './App.vue'
createApp(App).mount('#app')
```

### `src/App.vue` — 主组件

核心业务逻辑集中在单文件组件中，包含三部分：

#### `<script setup>` — 状态 & 方法

| 状态 | 类型 | 说明 |
|---|---|---|
| `messages` | ref | 聊天消息列表 `[{role, content}, ...]` |
| `inputValue` | ref | 输入框内容 |
| `loading` | ref | 加载状态 |
| `mode` | ref | 模式切换：`chat` / `search` |
| `searchResults` | ref | 检索模式结果 |
| `SESSION_ID` | const | 页面会话 ID（随机生成） |
| `DEFAULT_USER_CONTEXT` | const | 默认用户上下文 |

| 方法 | 说明 |
|---|---|
| `sendMessage()` | 对话模式：POST /api/chat，显示回答 |
| `sendSearch()` | 检索模式：POST /api/rag/search，显示卡片 |
| `toggleMode()` | 切换对话/检索模式 |
| `scrollToBottom()` | 自动滚动到底部 |

#### `<template>` — 界面

- **Header**：模式切换按钮（对话 / 检索）+ 状态指示灯
- **Body**：对话气泡 或 检索结果卡片
- **Footer**：输入框 + 发送按钮（Enter 发送，Shift+Enter 换行）

#### `<style scoped>` — 样式

- 玻璃拟态卡片
- 渐变色气泡
- 响应式布局

## 数据流

```
用户输入
  ↓
sendMessage()
  ↓
POST /api/chat  { message, history, session_id, user_context }
  ↓
等待 JSON 响应
  ↓
messages.push({ role: 'assistant', content: data.answer })
  ↓
Vue 响应式渲染 → 气泡显示
```

## 调用接口

| 模式 | 接口 | 请求方式 |
|---|---|---|
| 对话 | `/api/chat` | POST → JSON |
| 检索 | `/api/rag/search` | POST → JSON |
