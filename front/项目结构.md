---

## 核心文件

### `vite.config.js`

```javascript
server: { proxy: { '/api': { target: 'http://127.0.0.1:8000' } } }
```

### `src/App.vue`

单文件组件，包含三部分：

#### 状态管理

| 变量 | 类型 | 说明 |
|---|---|---|
| `messages` | `ref([])` | 聊天记录，刷新丢失 |
| `mode` | `ref('chat')` | 当前模式：chat_only / chat / search |
| `searchResults` | `ref([])` | 检索结果 |
| `loading` | `ref(false)` | 加载状态 |
| `SESSION_ID` | `const` | 页面会话 ID（随机） |
| `DEFAULT_USER_CONTEXT` | `const` | `{user_id:'1004', city:'北京', month}` |

#### 三种模式

| 模式 | 接口 | 标题 | 问候语 |
|---|---|---|---|
| 仅聊天 | `/api/chat` { mode: "chat_only" } | 小扫 · 聊聊天 | 嗨～我是小扫... |
| 智能问答 | `/api/chat` { mode: "chat" } | 扫地机器人智能客服 | 你好，我是扫地机器人... |
| 知识库检索 | `/api/rag/search` | 知识库检索 | 输入关键词搜索... |

通过 `<select>` 下拉框切换，切换时清空历史。

#### 核心方法

| 方法 | 说明 |
|---|---|
| `sendMessage()` | 对话模式，拼接最近 15 条 history 发 POST |
| `sendSearch()` | 检索模式，发 POST 显示结果卡片 |
| `switchMode(m)` | 切换模式 + 清空消息 + 设置问候语 |
| `scrollToBottom()` | 自动滚到底部 |

## 数据流

```
用户输入 → sendMessage()
  → history = messages.slice(-16, -1)
  → POST /api/chat
  → await response.json()
  → messages.push({ role:'assistant', content: data.data.answer })
```

## 调用接口

| 前端操作 | 接口 | 方式 |
|---|---|---|
| 仅聊天 / 智能问答 | `/api/chat` | POST → JSON |
| 知识库检索 | `/api/rag/search` | POST → JSON |
