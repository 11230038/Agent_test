# 前端项目结构

## 技术栈

- **Vue 3.5**（Composition API + `<script setup>`）
- **Vue Router 4**（Hash-less History 模式）
- **Vite 7**（开发 + 构建，含 API 代理）
- **marked** + **DOMPurify**（Markdown 渲染 + XSS 防护）

## 路由

| 路径 | 组件 | 说明 |
|---|---|---|
| `/` | ChatView | 聊天页面（仅聊天 / 智能问答 / 知识库检索） |
| `/admin` | AdminView | 管理页面（密码门 + 知识库增删改查） |

## 页面详情

### ChatView.vue — 聊天页

**三种模式：**

| 模式 | 接口 | 说明 |
|---|---|---|
| `chat_only`（仅聊天） | `POST /api/chat/stream` | SSE 流式，纯对话 |
| `chat`（智能问答） | `POST /api/chat/stream` | SSE 流式，RAG + 9 工具 |
| `search`（知识库检索） | `POST /api/rag/search` | JSON 一次性返回 |
| 自定义模式 | `POST /api/chat/stream` | SSE 流式，LLM 生成 Prompt + 9 工具 |

**核心逻辑：**
- 对话模式使用 SSE 流式接收，AbortController 支持中途停止
- 停止后已有内容追加"⏹ 输出已被终止"标记
- 「＋ 新对话」按钮：生成新 session_id + 清空消息（保持当前模式）
- Markdown 渲染：`marked` 解析 + `DOMPurify` 消毒 + GitHub 风格 CSS
- 检索模式展示结果卡片（来源、分类、相关性分值）
- 自动拼接最近 15 条 history
- 停止按钮：生成中显示红色脉冲"⏹ 停止生成"

### AdminView.vue — 管理页

**架构：** 10 行 `<script setup>`，全部逻辑委托给 4 个 composable

| Composable | 职责 |
|---|---|
| `useAdminAuth` | 登录 / JWT token 管理 |
| `useKnowledgeBase` | structure + 上传 + 删除 + 改分类 |
| `useChunkEditor` | Chunk 检索 + 编辑 |
| `usePromptEditor` | 提示词列表 / 读取 / 写入 |

**管理功能：**

| 功能 | 操作 | 接口 |
|---|---|---|
| 🔐 登录 | 输入密码 → 获取 JWT token | `POST /api/admin/login` |
| 📊 结构概览 | 显示总 Chunk / 分类数 / 追踪文件数 | `POST /api/admin/knowledge` |
| 📤 上传文档 | 选 PDF/TXT + 分类 → 上传 | `POST /api/admin/upload` |
| 🗑️ 删除文档 | 点击垃圾桶 → 弹窗确认 → 删除 | `POST /api/admin/delete` |
| 📁 改分类 | 点击文件夹图标 → 弹窗选分类 → 确认 | `POST /api/admin/category` |
| 🔍 检索 Chunk | 输入关键词实时过滤，展示匹配 chunk | 客户端过滤 |
| ✏️ 编辑 Chunk | 点击任意 chunk → 弹窗编辑 → 保存 | `POST /api/admin/chunk/update` |
| 📝 提示词 | 点击提示词文件 → 弹窗编辑 → 保存 | `POST /api/admin/prompts` 等 |

**交互细节：**
- 删除、改分类、提示词编辑均有确认弹窗（遮罩层 + 取消/确认按钮）
- Chunk 编辑和提示词编辑为独立宽弹窗
- 搜索有 250ms 防抖，结果以扁平列表展示
- JWT token 24h 过期，重启服务不丢失登录态

## 数据流

```
用户操作
  │
  ├─ 新对话 ──→ 新 sessionId + 重置 messages
  │
  ├─ 聊天/仅聊天模式 ──→ POST /api/chat/stream (SSE)
  │                          │
  │                    逐 chunk 追加到 messages
  │                          │
  │                    ┌─ 自然完成 → 渲染 Markdown
  │                    └─ 点停止 → AbortError → 追加终止标记
  │
  ├─ 检索模式 ──→ POST /api/rag/search
  │                    │
  │              展示结果卡片列表
  │
  └─ 管理页 ──→ JWT 登录 → 管理接口
                    │
             上传 / 删除 / 改分类 / 编辑 chunk / 提示词
                    │
             自动刷新结构树
```

## 调用的接口汇总

| 接口 | 用途 |
|---|---|
| `POST /api/chat/stream` | 流式对话（AbortController 可中断） |
| `POST /api/rag/search` | 知识库检索 |
| `POST /api/admin/login` | 登录获取 JWT |
| `POST /api/admin/knowledge` | JWT 验证 + 获取结构 |
| `POST /api/admin/upload` | 上传文档 |
| `POST /api/admin/delete` | 删除文档 |
| `POST /api/admin/category` | 更改分类 |
| `POST /api/admin/chunk/update` | 编辑 chunk |
| `POST /api/admin/prompts` | 提示词列表 |
| `POST /api/admin/prompt/read` | 读取提示词 |
| `POST /api/admin/prompt/write` | 写入提示词 |
| `POST /api/admin/config` | 更新系统配置 |
| `POST /api/admin/mode/create` | 创建自定义模式 |
| `GET /api/modes` | 获取可用模式 |
| `GET /api/config` | 获取系统配置 |
| `GET /api/sessions` | 历史会话列表 |
