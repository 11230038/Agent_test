# 后端 API 接口文档

## 通用约定

### 响应格式

```json
{ "success": true | false, "message": "说明", "data": { ... } | null }
```

### 错误状态码

| 状态码 | 含义 | 对外文案 |
|---|---|---|
| 200 | 成功 | — |
| 400 | 请求参数错误 | 参数错误具体原因 |
| 422 | 请求格式校验失败 | "请求参数校验失败" |
| 502 | 模型执行失败 | "模型服务暂不可用，请稍后重试" |
| 503 | 配置/依赖异常 | "服务配置异常，请联系管理员" |
| 500 | 未知错误 | "服务器内部错误，请稍后重试" |

## 一、系统接口

### `GET /`
服务状态。

```json
{ "success": true, "message": "ok", "data": { "message": "Robot Agent API is running" } }
```

### `GET /health`
进程存活检查，不依赖外部服务，始终返回 200。

```json
{ "success": true, "message": "ok", "data": { "status": "ok" } }
```

### `GET /ready`
依赖就绪检查，检测 LLM + Embedding + Chroma 连通性。

```json
{
  "success": true,
  "message": "ok",
  "data": {
    "checks": {
      "llm": { "ok": true },
      "embedding": { "ok": true },
      "chroma": { "ok": true, "chunk_count": 318 }
    }
  }
}
```

有依赖异常时 `message` 会标注：`"degraded: llm, chroma 不可用"`。

## 二、对话接口

### `POST /api/chat`（非流式）

**请求体：**

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `message` | string | 是 | 当前用户问题 |
| `history` | array | 否 | 历史消息 `[{role, content}]` |
| `session_id` | string | 否 | 会话标识，用于持久化和追踪 |
| `user_context` | object | 否 | `{user_id, city, month}` |
| `mode` | string | 否 | `chat`（默认）/ `chat_only` / `report` |

**请求示例：**
```json
{
  "message": "如何更换滤网",
  "history": [
    { "role": "user", "content": "你好" },
    { "role": "assistant", "content": "你好！有什么可以帮你的？" }
  ],
  "session_id": "sess_abc123",
  "user_context": { "user_id": "1004", "city": "北京", "month": "2026-06" },
  "mode": "chat"
}
```

**响应示例：**
```json
{
  "success": true,
  "message": "请求成功",
  "data": {
    "answer": "滤网更换步骤如下...参考来源：[1] 维护保养.txt",
    "trace_id": "a1b2c3d4e5f6",
    "sources": [
      { "source": "维护保养.txt", "category": "维护保养", "score": 0.714 }
    ]
  }
}
```

**副作用：**
- 自动保存对话到 session 存储
- 自动提取用户画像（偏好 + 性别 + 年龄）
- 自动保存 RAG 来源引用

### `POST /api/chat/stream`（SSE 流式）

请求体同 `/api/chat`，响应为 SSE 格式：

```
data: 滤网
data: 更换步骤如下
data: （空行，心跳保活）
data: ：1. 关闭电源...
```

- 4 秒无新内容时发送空心跳
- 流自然结束即完成
- 错误时发送 `data: [ERROR] 错误信息`

### mode 说明

| mode | Agent | 工具 | Prompt | 适用场景 |
|---|---|---|---|---|
| `chat` | 主 Agent | 9 个工具 | main_prompt.txt | 产品咨询、故障排查、报告 |
| `chat_only` | 聊天 Agent | 无 | chat_prompt.txt | 闲聊、打招呼 |
| `report` | 主 Agent | 9 个工具 | 自动切换 report_prompt.txt | 生成使用报告 |

## 三、知识库接口

### `POST /api/rag/search`

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `query` | string | 是 | 检索关键词 |
| `style` | string | 否 | 格式化风格：`sources`（默认）/ `full` / `compact` |

```json
{
  "success": true,
  "data": {
    "query": "WIFI设置",
    "total": 2,
    "results": [
      {
        "content": "在APP【设置】-【消息提醒】中...",
        "score": 1.128,
        "source": "扫拖一体机器人100问.txt",
        "category": "扫拖一体",
        "metadata": { "category": "扫拖一体", "source": "..." }
      }
    ]
  }
}
```

### `GET /api/rag/status`

```json
{
  "success": true,
  "data": {
    "knowledge_base": { "chunk_count": 318, "categories": [...], "collection_name": "agent" },
    "cache": { "hits": 12, "misses": 45, "size": 8, "hit_rate": 0.211 },
    "embedding": { "available": true, "error": null },
    "retrieval": { "k": 3, "similarity_threshold": 0.3 }
  }
}
```

## 四、会话接口

### `GET /api/sessions`
列出所有会话，按最后活跃时间倒序。

```json
{
  "success": true,
  "data": {
    "sessions": [
      { "session_id": "sess_abc", "msg_count": 12, "last_active": 1781246374.92 }
    ]
  }
}
```

### `GET /api/session/{session_id}`
获取指定会话的完整历史。

```json
{
  "success": true,
  "data": {
    "session_id": "sess_abc",
    "history": [
      { "role": "user", "content": "你好" },
      { "role": "assistant", "content": "你好！" }
    ]
  }
}
```

### `DELETE /api/session/{session_id}`
删除指定会话的所有消息。

## 五、用户画像接口

### `POST /api/profile`

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `user_id` | string | 是 | 用户唯一标识 |
| `city` | string | 否 | 城市 |
| `gender` | string | 否 | 性别（男/女） |
| `age` | int | 否 | 年龄 |
| `preferences` | object | 否 | 偏好键值对 |

> user_id 为"未知用户"时返回 400

### `GET /api/profile/{user_id}`

```json
{
  "success": true,
  "data": {
    "profile": {
      "user_id": "1004",
      "city": "北京",
      "gender": "男",
      "age": 35,
      "preferences": { "house_type": "大户型", "has_pets": true },
      "updated_at": 1781246370.98
    }
  }
}
```

## 六、反馈接口

### `POST /api/feedback`

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `session_id` | string | 是 | 会话标识 |
| `rating` | string | 是 | `like` 或 `dislike` |
| `comment` | string | 否 | 反馈备注 |

### `GET /api/feedback/stats`

```json
{
  "success": true,
  "data": {
    "total": 128,
    "likes": 95,
    "dislikes": 33,
    "recent": [
      { "session_id": "sess_abc", "rating": "like", "comment": "回答准确", "created_at": 1781247325.34 }
    ]
  }
}
```
